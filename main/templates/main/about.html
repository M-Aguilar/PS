This page contains information about the site including software, hardware, and other relevant materials used in setting up and executing this site. Feel free to contact me for any information regarding any project or the site itself via email.

Software

Raspbian

Raspbian is the recommended operating system for the raspberry pi. It can be used as a personal computer by connecting various peripherals including mouse, keyboard, and display. For my purposes I remote log in with a different computer or phone.

Python

Python is included in Raspbian and is used for various purposes such as create virtual environments and running simple scripts as well as designing the functionality of this site.

Django

Django is a python based framework that provides a complete option for developing a functional website. A different python based alternaitve to Django is Flask.

Bootstrap

Bootstrap is a styling solution. Using jQuery to provide clean functional website elements.

Sublime Text Editor

Sublime is my preferred text editor. it is simple and small in size. It supports most languages and can be customized extensively as desired.

Apache

Apache is the most popular web serving framework or so I here and is a key piece in hosting a website.

Hardware

Raspberry pi

My main file sites are stored on my raspberry pi 2 model B+. This is a great tool for learning basic computer functionality as well as an introduction to more technical skills such as programming, command line usage, and useful computer practices. Some great projects include setting up and hosting a site, VPN, and NAS server.
The raspberry pi was released on June 24, 2019 and can be used for a lot. Older models are also still viable today for most purposes.
